Console Game Play
=================

loop1:
    select X player (human or some type of computer player)
    select O player (human or some type of computer player)
    load computer players
loop2:
    Play One Game
    ask action:
    - quit
    - same players and pieces
    - same players and diff pieces
    - diff players
    if quit, exit
    if same players and pieces, goto loop2
    if same players and diff pieces:
        swap players
        goto loop2
    goto loop1

Play One Game
-------------
current player = X
reset board
reset players
while winner is none:
    indicate current player
    show board
    get move from current player
    announce move
    update board with move
    switch current player to opposite piece
show board
display winner
